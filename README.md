# Capstone 2: Business Analysis with Python

## Overview
This project analyzes sales data for **EmporiUm**, a fictional student bookstore chain selling books, tech gear, art supplies, and more — both in-store and online. The goal is to compare sales performance across two assigned sales territories and deliver actionable insights to the marketing team.



## Analysis Summary
The Jupyter Notebook walks through:
1. **Setup & Data Loading** – Importing libraries and reading in sales data
2. **Data Cleaning** – Handling missing values, formatting, and filtering
3. **Exploratory Analysis** – Summary statistics and initial observations
4. **Territory Comparison** – Side-by-side analysis of the two assigned sales territories
5. **Visualizations** – At least two Matplotlib charts illustrating key findings
6. **Core Marketing Analysis** – Insights and recommendations for the marketing team

## Tools & Libraries
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook
